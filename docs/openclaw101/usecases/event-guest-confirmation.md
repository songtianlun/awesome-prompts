# 活动嘉宾确认

你在举办活动 —— 晚宴、婚礼、公司外出 —— 需要确认嘉宾出席。手动打电话给 20+ 个人很枯燥：你和人打电话标签游戏，忘记谁说了什么，丢失饮食要求或加一信息。发短信有时有用，但人们会忽略消息。真正的电话通话能获得高得多的回复率。

这个用例让 OpenClaw 使用 [SuperCall](https://clawhub.ai/xonder/supercall) 插件给你名单上的每个嘉宾打电话，确认是否参加，收集备注，并为你编译摘要。

## 功能

- 遍历嘉宾名单（姓名 + 电话号码）并逐一打电话
- AI 以友好的活动协调员身份自我介绍
- 确认活动日期、时间和地点
- 询问是否参加，并收集备注（饮食需求、加一、到达时间等）
- 所有通话完成后，编译摘要：谁确认了、谁拒绝了、谁没接电话、各种备注

## 为什么选择 SuperCall

SuperCall 是完全独立的语音智能体。通话中的 AI 角色**只能访问你提供的上下文**（角色名、目标和开场白）。它无法访问你的网关智能体、文件、其他工具或任何其他内容。

这对嘉宾确认很重要：

- **安全性**：通话另一端的人无法通过对话操纵或访问你的智能体
- **更好的对话**：因为 AI 专注于单一任务（确认出席），它保持主题且对话更自然
- **适合批量**：你要给不同的人打很多电话。每次通话重置的沙盒角色正是你需要的

## 所需技能

- [SuperCall](https://clawhub.ai/xonder/supercall) —— 通过 `openclaw plugins install @xonder/supercall` 安装
- Twilio 账号和电话号码（用于拨出电话）
- OpenAI API 密钥（用于 GPT-4o Realtime 语音 AI）
- ngrok（用于 webhook 隧道）

## 如何设置

1. 准备嘉宾名单：

```text
嘉宾名单 — 夏季烧烤，6月14日周六，下午4点，橡树街23号

- 张三：+8613800138000
- 李四：+8613900139000
- 王五：+8613700137000
```

2. 向 OpenClaw 发送：

```text
我需要你确认我的活动出席情况。以下是详情：

活动：夏季烧烤
日期：6月14日周六下午4点
地点：橡树街23号

这是我的嘉宾名单：
<在此粘贴嘉宾名单>

对每位嘉宾，使用 supercall 打电话。使用"Jamie，[你的名字]的活动协调员"
作为角色。每次通话的目标是确认他们是否参加，并记下饮食限制、加一或其他备注。

每次通话后记录结果。所有通话完成后，给我完整摘要：
- 谁确认了
- 谁拒绝了
- 谁没接电话
- 每位嘉宾的备注或特殊要求
```

## 关键洞察

- **先小规模测试**：先对 2-3 位嘉宾试验，确保角色和开场白听起来合适。
- **注意拨打时间**：不要安排太早或太晚的电话。
- **真实电话通话需要费用**：每次通话使用 Twilio 通话分钟。在 Twilio 账户中设置适当的限额。
