# 多智能体内容工厂

你是一个内容创作者，同时处理多个平台的研究、写作和设计。每个步骤 —— 找热门话题、写脚本、生成缩略图 —— 都要花费数小时。如果一个专业化的智能体团队在一夜之间处理所有这些呢？

这个工作流在 Discord 中建立一个多智能体内容工厂，不同的智能体在专用频道中处理研究、写作和视觉资产。

## 功能

- **研究智能体** 每天早上扫描热门故事、竞争对手内容和社交媒体，找出最佳内容机会
- **写作智能体** 选取最好的创意，撰写完整的脚本、推文线程或新闻简报草稿
- **缩略图智能体** 为内容生成 AI 缩略图或封面图片
- 每个智能体在自己的 Discord 频道中工作，保持一切有序且可审查
- 按计划自动运行（例如每天早上 8 点），让你醒来就看到完成的内容

## 痛点

内容创作有三个阶段 —— 研究、写作和设计 —— 大多数创作者在手动完成所有三个阶段。即使有 AI 写作工具，你仍然需要一个一个地提示它们。这个系统将智能体串联成一个流水线，一个智能体的输出馈入下一个，完全不需要人工干预。

## 所需技能

- Discord 集成，配置多个频道
- `sessions_spawn` / `sessions_send` 用于多智能体编排
- [x-research-v2](https://clawhub.ai) 或类似社交媒体研究工具
- 本地图像生成（如 Nano Banana）或图像生成 API
- [knowledge-base](https://clawhub.ai) 技能（可选，用于 RAG 驱动的研究）

## 如何设置

1. 设置一个 Discord 服务器（或让 OpenClaw 帮你创建 —— 直接说"为我们建一个 Discord"）。

2. 为每个智能体创建频道：
   - `#research` —— 热门话题和内容机会
   - `#scripts` —— 写作草稿和大纲
   - `#thumbnails` —— 生成的图片和封面

3. 向 OpenClaw 发送：

```text
我希望你在 Discord 中为我建一个内容工厂。
为不同的智能体设置频道：

1. 研究智能体（#research）：每天早上 8 点，研究最热门的
   故事、竞争对手内容以及社交媒体上在我的领域中表现好的内容。
   发布前 5 个内容机会并附上来源。

2. 写作智能体（#scripts）：选取研究智能体的最佳创意，
   撰写完整的脚本/推文线程/新闻简报草稿。发布到 #scripts。

3. 缩略图智能体（#thumbnails）：为内容生成 AI 缩略图或封面图片。
   发布到 #thumbnails。

将所有工作组织在不同的频道中。
每天早上自动运行此流水线。
```

4. 为你的平台定制：

```text
我专注于 X/Twitter 推文线程，而非 YouTube。请将写作智能体
改为生成推文线程而非视频脚本。
```

## 关键洞察

- 力量在于**串联的智能体** —— 研究馈入写作，写作馈入缩略图。你不需要单独提示每个步骤。
- Discord 频道让审查每个智能体的工作变得简单，你可以反馈"脚本太长了"或"多关注 AI 新闻"。
- 你可以将其适配为任何内容格式：推文、新闻简报、LinkedIn 帖子、播客大纲、博客文章。
- 使用本地模型进行图像生成（如在 Mac Studio 上运行 Nano Banana）可以降低成本并给你更多控制权。
