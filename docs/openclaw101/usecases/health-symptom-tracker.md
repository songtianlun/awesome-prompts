# 健康与症状追踪器

识别食物敏感性需要长期持续的记录，这很枯燥。你需要提醒来记录，还需要分析来发现模式。

这个工作流自动追踪食物和症状：

- 在专用的 Telegram 主题中发送你的食物和症状，OpenClaw 会自动记录时间戳
- 每日 3 次提醒（早、中、晚）提示你记录饮食
- 随着时间推移，分析模式以识别潜在诱因

## 所需技能

- 定时任务用于提醒
- Telegram 主题用于记录
- 文件存储（markdown 日志文件）

## 如何设置

1. 创建一个名为 "health-tracker" 的 Telegram 主题。
2. 创建日志文件：`~/clawd/memory/health-log.md`
3. 向 OpenClaw 发送：

```text
当我在 "health-tracker" 主题中发消息时：
1. 解析消息中的食物和症状
2. 带时间戳记录到 ~/clawd/memory/health-log.md
3. 确认记录了什么

设置 3 个每日提醒：
- 上午 8 点："记录你的早餐"
- 下午 1 点："记录你的午餐"
- 晚上 7 点："记录你的晚餐和任何症状"

每个周日，分析过去一周的日志并识别模式：
- 哪些食物与症状相关？
- 是否有时间规律？
- 有明确的诱因吗？

将分析发布到 health-tracker 主题。
```

4. 可选：添加一个记忆文件，让 OpenClaw 追踪已知诱因，并在新模式出现时更新。
