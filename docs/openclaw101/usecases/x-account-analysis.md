# X 账号分析

有很多网站可以对你的 X 账号进行定性分析。虽然 X 已经有了**分析**板块，但它更侧重于展示你的数据表现。

而定性分析关注的是你帖子的**质量**，而非表现数据。你可以从这种分析中获得的洞察包括：

- 什么模式让我的帖子走红？
- 我讨论哪些话题能获得最多互动？
- 为什么有些帖子能获得 1000+ 点赞，而有些帖子不到 5 个赞？我做错了什么？

市面上有很多网站和应用提供 X 分析，但它们只关注统计数据。大概只有 1-2 个网站允许你与 AI 对话来理解你的表现。

现在你可以用 OpenClaw 来完成这个分析，无需在这些网站上花 10-50 美元订阅。

## 所需技能

Bird 技能。`clawhub install bird`（预装）

## 如何设置

流程如下：

1. 确保 Bird 技能正常工作。
2. 为了安全和隔离，最好为你的 ClawdBot 创建一个新账号。
3. 用你的 X 账号认证。在 Chrome/Brave 中登录 x.com，提供正确的 cookie 信息（`auth-token`、`ct0`），这样 OpenClaw 就能访问你的账号。
4. 让 OpenClaw 查看你的真实账号，获取最近 N 条推文，然后问它任何你想了解的问题。或者，你可以让它为你编写特定的分析脚本。
