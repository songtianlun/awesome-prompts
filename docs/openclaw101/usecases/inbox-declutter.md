# 收件箱整理

新闻简报可以像洪水一样淹没你的收件箱。它们经常堆积如山，根本没有被打开过。

## 所需技能

[Gmail OAuth 设置](https://clawhub.ai/kai-jar/gmail-oauth)。

## 如何设置

1. [可选] 专门为 OpenClaw 创建一个新的 Gmail 账号。
2. [可选] 取消主邮箱的所有新闻简报订阅，改用 OpenClaw 的邮箱订阅。
3. 安装技能并确保其正常工作。
4. 指示 OpenClaw：

```text
我希望你每天晚上 8 点运行定时任务，阅读过去 24 小时的所有新闻简报邮件，给我一份最重要内容的摘要，并附上阅读更多的链接。然后询问我对你挑选的内容的反馈，并根据我的偏好更新你的记忆，以便在未来的任务中提供更好的摘要。
```
