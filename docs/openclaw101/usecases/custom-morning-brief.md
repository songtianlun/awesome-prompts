# 定制早间简报

你醒来后花 30 分钟追赶信息 —— 刷新闻、查日历、看待办清单、试图弄清今天什么重要。如果这些都已经完成并以短信形式等着你呢？

这个工作流让 OpenClaw 每天在固定时间向你发送完全定制的早间简报，涵盖新闻、任务、创意和主动推荐。

## 功能

- 每天同一时间（如上午 8:00）向 Telegram、Discord 或 iMessage 发送结构化的早间报告
- 通过浏览网页研究与你兴趣相关的隔夜新闻
- 审查你的待办清单，呈现当天的任务
- 在你睡觉时生成创作性输出（完整脚本、邮件草稿、商业提案 —— 不仅是创意）
- 推荐 AI 可以自主完成的任务来帮助你

## 痛点

你在最高效的早晨时间里只是在定位方向。同时，你的 AI 智能体整夜闲置。早间简报将空闲的夜间时间变成高效的准备时间 —— 你醒来时工作已经完成了。

## 所需技能

- Telegram、Discord 或 iMessage 集成
- Todoist / Apple Reminders / Asana 集成（你使用的任务管理工具）
- [x-research-v2](https://clawhub.ai) 用于社交媒体趋势研究（可选）

## 如何设置

1. 将 OpenClaw 连接到你的消息平台和任务管理器。

2. 向 OpenClaw 发送：

```text
我想设置一个每日早间简报。每天早上 8:00 通过 Telegram 发送报告。

我希望这个报告包含：
1. 与我兴趣相关的新闻（AI、创业、科技）
2. 今天可以创作的内容创意
3. 今天需要完成的任务（从我的待办清单拉取）
4. 你今天可以替我完成的任务推荐

对于内容创意，写完整的草稿脚本/大纲 —— 不只是标题。
```

3. 随时间定制 —— 直接发消息给你的机器人：

```text
在我的早间简报中加入天气预报。
停止包含一般新闻，只关注 AI。
每天早上包含一句励志名言。
```

## 关键洞察

- AI 推荐任务部分是最强大的 —— 它让智能体主动思考如何帮助你，而非等待指令。
- 你可以通过发消息定制简报。说"在我的早间简报中加入股票价格"就会更新。
- 完整草稿（而非仅创意）是节省时间的关键。醒来看到脚本，而非建议。
