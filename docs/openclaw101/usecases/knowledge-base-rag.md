# 个人知识库（RAG）

你整天阅读文章、推文、看视频，但永远找不到上周看到的那个东西。书签堆积如山变得毫无用处。

这个工作流从你保存的一切构建可搜索的知识库：

- 在 Telegram 或 Slack 中放入任何 URL，它会自动消化内容（文章、推文、YouTube 字幕、PDF）
- 对你保存的一切进行语义搜索："我保存了什么关于智能体记忆的？"返回排名结果和来源
- 馈入其他工作流 —— 例如视频创意流水线在构建研究卡片时查询知识库获取相关已保存内容

## 所需技能

- [knowledge-base](https://clawhub.ai) 技能（或用嵌入构建自定义 RAG）
- `web_fetch`（内置）
- Telegram 主题或 Slack 频道用于录入

## 如何设置

1. 从 ClawdHub 安装 knowledge-base 技能。
2. 创建一个名为 "knowledge-base" 的 Telegram 主题（或使用 Slack 频道）。
3. 向 OpenClaw 发送：

```text
当我在 "knowledge-base" 主题中放入 URL 时：
1. 获取内容（文章、推文、YouTube 字幕、PDF）
2. 将其录入知识库，包含元数据（标题、URL、日期、类型）
3. 回复确认：录入了什么以及分块数量

当我在此主题中提问时：
1. 语义搜索知识库
2. 返回排名最高的结果，包含来源和相关摘录
3. 如果没有好的匹配，告诉我

另外：当其他工作流需要研究时（如视频创意、会议准备），自动查询知识库获取相关已保存内容。
```

4. 测试：放入一些 URL，然后问"我有什么关于 LLM 记忆的内容？"
