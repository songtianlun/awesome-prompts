# AI 财报追踪器

追踪数十家科技公司的财报季意味着检查多个信息源和记住报告日期。你想紧跟 AI/科技财报而不必手动追踪每家公司。

这个工作流自动化了财报追踪和推送：

- 每周日预览：扫描即将到来的财报日历，将相关科技/AI 公司发布到 Telegram
- 你选择关注哪些公司，OpenClaw 为每个财报日期安排一次性定时任务
- 每份报告发布后，OpenClaw 搜索结果，格式化详细摘要（超预期/不及预期、关键指标、AI 亮点），并推送

## 所需技能

- `web_search`（内置）
- OpenClaw 定时任务支持
- Telegram 主题用于财报更新

## 如何设置

1. 创建一个名为 "earnings" 的 Telegram 主题。

2. 向 OpenClaw 发送：

```text
每周日下午 6 点运行定时任务：
1. 搜索下周科技和 AI 公司的财报日历
2. 过滤我关注的公司（NVDA、MSFT、GOOGL、META、AMZN、TSLA、AMD 等）
3. 将列表发布到我的 Telegram "earnings" 主题
4. 等待我确认要追踪哪些

当我回复要追踪的公司时：
1. 为每个财报日期/时间安排一次性定时任务
2. 报告发布后，搜索财报结果
3. 格式化摘要，包含：超/不及预期、收入、每股收益、关键指标、AI 相关亮点、前瞻指引
4. 发布到 Telegram "earnings" 主题

在记忆中保存我通常追踪的公司，以便每周自动推荐。
```
